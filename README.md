# E-Commerce-Sales-Analytics-Report
Power BI Report for analyzing e-commerce sales data to uncover revenue trends, product performance, and regional insights using interactive visualizations

## Project Overview
The **E-Commerce Sales Analytics Dashboard** is an interactive business intelligence report built using **Power BI** to analyze online sales performance.

This dashboard provides insights into **sales revenue, product performance, regional contribution, and monthly trends**, helping businesses understand customer purchasing behavior and optimize sales strategies.

The report transforms raw transactional data into **meaningful visual insights** to support **data-driven decision making**.

---

## Objective
The main objective of this project is to analyze **online sales data** and present key insights through an interactive dashboard.

Key goals include:

- Monitor **total revenue and sales performance**
- Identify **top performing products**
- Analyze **sales trends across months**
- Compare **revenue contribution by region**
- Understand **product category performance**
- Enable **interactive filtering for deeper analysis**

---

## Dataset Information

The dataset contains transactional data from an e-commerce platform.

### Dataset Columns

| Column Name | Description |
|-------------|-------------|
| Transaction ID | Unique ID for each order |
| Date | Transaction date |
| Product Category | Category of the product |
| Product Name | Name of the product |
| Units Sold | Number of units purchased |
| Unit Price | Price per unit |
| Total Revenue | Revenue generated from the transaction |
| Region | Sales region (North America, Asia, Europe) |
| Payment Method | Mode of payment used by customer |

## Dashboard Configuration Details

### Data Preparation
- Imported raw dataset into Power BI
- Cleaned data using **Power Query**
- Checked for missing or duplicate values
- Ensured correct data types for each column

### Data Modeling
Created calculated measures using DAX:

```DAX
Total Revenue = SUM(Sales[Total Revenue])

Total Units Sold = SUM(Sales[Units Sold])

Total Transactions = COUNT(Sales[Transaction ID])
```

### Visualization Design
- KPI Cards for quick insights
- Line chart for monthly trend analysis
- Bar charts for product performance
- Donut chart for regional comparison
- Slicers for user-driven exploration
